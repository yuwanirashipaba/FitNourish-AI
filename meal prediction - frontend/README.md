# Meal Prediction Frontend

React frontend for meal prediction application built with Vite.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or another port if 5173 is occupied)

## Features

- **Left Panel**: Upload meal images and analyze ingredients, nutrients, and calories
- **Right Panel**: Get personalized meal suggestions based on daily calorie goals

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Requirements

- Node.js v20.12.2 or higher
- npm 10.5.0 or higher

Make sure the backend API is running on `http://localhost:8000` for the frontend to work properly.
